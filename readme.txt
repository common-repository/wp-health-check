WP Health Check scans your WordPress installation and reports any security issues with your site. 

== Description ==

WP Health Check scans your WordPress installation and reports any security issues with your site. It will also recommend you the solution to ovecome the issues. The purpose of this plugin is to keep your WordPress installation Safe and Secure from hackers.

Features in WP Health Check 1.1 include:

* Checks if you are using the latest version of WordPress
* Checks if you are using the latest version of WordPress Plugins
* Checks if you are using the latest version of WordPress Themes
* Checks if it is possible to list the contents of the Uploads, Plugins or Themes directory
* Checks if you have any inactive plugins which needs to be deleted
* Checks if there is any user with the username "admin"
* Checks if you are using the database prefix as "wp_"
* Completely FREE
* Licensed under GNU GPL version 3
* Safe & Secure


== Installation ==

Upload the WP Health Check to your blog, Activate it.
That's it. You're done!
